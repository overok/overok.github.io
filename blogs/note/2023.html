2023总结

over2023/07/20

# 五月

也不知道素材断了还是我脑子健忘了，2023到现在一片空白。记录一下最近发生的事吧。目前我还在傲鹰这家公司工作，主要工作太过轻松了，不想跳出这个舒适圈，所以一直摆烂，打算一直这样拖到考编。
1.小关回家考编了
2.五一参加了建平的婚礼
3.代为民和刘红也快结婚了
4.我也打算下半年和小关订婚及买房了
5.独立自己+健身

# 一些焦虑

回顾2023的上半年，感觉生活并没有多大起色，一切平平淡淡。但最近有些事也着实让我感到心烦，公司安排整个技术部都出去执行表演，这其中的劳累程度也自然不必多说。所以心里萌启了离职的想法，想来也是来这家公司快俩年了，也该离开舒适圈了，像以前一样。根据目前市场行情，离职这件事当然要从长计议，绝不可断了生活来源，给自己制造焦虑。

# 独处

最近这段时间小关回家了，一个人感觉在上海挺无聊的，幸好米苏苏来了，让我枯燥的生活带来了一些趣味。 最近在学习PS及达芬奇，晚上也在健身，想要把身材锻炼出来。总之我一直在向好的方向上努力进步，期待六月份和小关见面时，我会变得更加优秀。

# 主线任务

主线任务考编制不能断，这是最终目标，但考虑到我的本科证需要明年才能发下来，今年下半年也没有一些合适的考试岗位，所以这段过渡时间还需要坚持，以挣钱为主，当前大形势多存点money。

# 人生如梦

昨晚看了《白日梦想家》，给我带来挺多感触的，男主的说走就走的性格挺像我几年前的，但不知何时我慢慢被生活磨平了，害，还是等考公上岸吧。或许到时候我的压力和焦虑会小一些吧，到时候带着小关一起去享受生活的自由。所以，现在，奋斗当下。

# 分手

最近被感情的事烦透了头脑，双方家庭的矛盾，我和小关之间的矛盾，似乎都发展到不可控的地步，我也不清楚到底什么在阻碍我们前行。国庆节给小关买了黄金，然后小关提了分手，我也身心疲惫，这离订婚的路似乎越来越远了。小关10.15过生日，买的生日礼物尚还未有机会送给她，按照目前的形势看，尚且没有必要了。 六年的感情就如此吧，既然都做出了选择，想必都是深思熟虑的，而我，得过且过吧，爱好老爸老妈老妹就足够了，感情的事我真的累了。

# 搬家

居住了快俩年的房子最近退了，搬到和米苏一起住了，一是确实离公司太远了，很不方便，二是感觉房子充满回忆，一个人比较孤单。在门口拍下这几张照片，这些陪伴了我刚来上海时居无定所，以及我和小关打打闹闹生活回忆，每次在一起都做一大桌美味饭菜，当然这些都是回忆了，迫于现实的状况，只能说再见了。

# 平安夜

今年的平安夜心情有些复杂，刚刚经历过分手，整个人感觉失去了动力，原本的计划目标也停滞下来了。陷入了孤独循环，今年暂定的目标可以说都未完成。可是生活还得继续，从下周一重新开始上路，只有自己变得够优秀，才不会有杂乱的思想，只管去做就对了。

# 年末回顾

今天是一月一日，我在路途上写下这份年度总结。
回顾这一年里，我仿佛没有什么高光点，更不要谈论一些有意义的事了，或许是本命年的缘故，我从头糟到尾，工作的焦虑，感情的焦虑以及思想的焦虑😐。这些焦虑无时无刻不在冲击着我的大脑，而我似乎也败倒它们面前。
今年也是我来上海的第二年，这俩年里我一直在傲鹰徘徊着，不知不觉青春的激情已经淡化，说走就走的决心以及执行力都在消失，我深知这样会让我迷失，让我沉沦，但我好像没有办法挣脱这无底的深渊，只能一味的得过且过。
感情的事也是如此，我从未想过我和小关会分手，一路走来，虽有大大小小的矛盾，但美好的回忆总是占大多数的。明明只差最后一步就该结婚了，但青春的遗憾就这样草草收尾了。
年底最后时刻，幸好有几个好朋友一直陪着我，米苏带给我说做就做的执行力，肉球带来给我为人处事的道理，我感谢并热爱他们。让我在生命的黑暗里寻找到一丝救赎自己的微光。
2023年度总结一句话：得之我幸，失之我命。
2024年度开始一句话：要做马背上的狂风，暴雨也浇不灭的烈火🔥
