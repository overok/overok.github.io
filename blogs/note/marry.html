# 婚姻

这段时间一直在和小关商讨婚姻相关的事宜，中间一直发生了一些不愉快的矛盾冲突。现在反思来看，主要还是自己问题。理智冲昏了我的头脑，一直站在利己的角度在考虑，完全没有设身处地的为小关着想，对此，我为我最爱的人深感愧疚与抱歉。写出这段话是我的想法，但不是我的本意。我向来对婚姻没有什么憧憬向往，并不是逃避现实，而是内在的精神束缚，在走向婚姻的这段路上我自认为还没有准备好，性格的不成熟以及思想的未改变远远达不到“丈夫”这个角色的要求。似乎还停留在热恋期阶段，不用考虑太多，只要和对方吃喝玩乐就一切满足。

我知道最终这一步是要到来，这不仅是对小关的负责，也是对自己的负责，一个新家庭的建立，寄托了俩家人的期望，婚后的生活也会更加稳定。虽知道这是必然的结果，但目前自己的思想还处于麻木阶段，似乎现在每走一步都像是提线木偶般的艰难，连最基本的计划和方向都没有。

再谈谈我和小关吧，我呢一直是个追求思想和生活自由的人，不喜欢被外在的条件所束缚，如果有的选，我更喜欢一个人安静的独处，这不是在逃避社会，而是来自精神的压抑。但我外在的表现却又显得格格相反，这不可笑吗，或许这是人格分裂吧。小关让我陪她打羽毛球，我的思想立马就抵触了起来，完全都没有思考的机会。我也不知道这啥毛病，其实我也不想这样，看，我连这么简单的要求都达不到，以后还怎么谈得上照顾家庭及对小关负责。小关哩，她一有时间就喜欢黏着我，这不一下陷入死循环BUG了吗，所以经常惹得小关生气，在这里我要和她说声抱歉。至于小关本人，我没什么好说的，满分100分。

或许如大多人所说，结完婚就会变成熟了，婚姻就是这样磕磕碰碰，对于这些说法我不做评论，但至少现在，我觉得我必须有所改变。这一步我要完美负责任的迈过去。

# 分手

最近被感情的事烦透了头脑，双方家庭的矛盾，我和小关之间的矛盾，似乎都发展到不可控的地步，我也不清楚到底什么在阻碍我们前行。国庆节给小关买了黄金，然后小关提了分手，我也身心疲惫，这离订婚的路似乎越来越远了。小关10.15过生日，买的生日礼物尚还未有机会送给她，按照目前的形势看，尚且没有必要了。 六年的感情就如此吧，既然都做出了选择，想必都是深思熟虑的，而我，得过且过吧，爱好老爸老妈老妹就足够了，感情的事我真的累了。
