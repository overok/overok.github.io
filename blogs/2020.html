# 2020总结

2020年12月份年底，我突然萌生了这样一个想法,把每天的生活和感悟记录一下，以便未来回首向往。

这件事其实计划了很久，但一直没有得到有效的执行，拖延症实在困扰着我。 这几年一直都感觉匆匆忙忙的，2018，2019时间转瞬即逝。以致于每年年尾回望过去，总感觉没有半点回忆，需要珍惜的有很多，但是自己总是放任他们离去。所以来一个每年计划安排，不为学习和目的，只为往后记忆。

# 至暗时刻

这个年的年底尾巴确实过的不怎么好，被好多羁绊困扰着，或许主要部分是恨自己的无能吧。匆匆二十余年载，回首的却都是悔恨。 世上美好的事物有大多，但我的眼界却一直局限于此。 这一年的生活没有太多的起色，从2020年年出到现在我一直处于一个迷茫的状态，工作，学习什么都不顺心，仿佛自己就是个失败者。生活没有乐趣，思想也落后，连一个精神支柱也没有,生活和人生的尽头全是黑暗。

# 前端

工作一年至今，我发现我个人性格虽然比较开放和喜欢交友，但对于工作上和别人交流是非常厌恶的，非常可恨销售那种类似工作方式，或许我的内心就是喜欢孤独的一个人存在。因此在6-7月的夏天，我靠别了碌碌无为的自媒体行业，转向了前端技术行业，至少我对计算机还是比较感兴趣的。初来乍到的滋味并不好，一个月只能拿个2700的实习工资，不分早晚的在小外包公司干着。因为是新人再加上工资微乎其微，以至于生活质量在逐渐下降，常常在夜里思考人生的挫败。还好写代码是一个比较沉浸式的工作，能够让我短暂的忘记生活上的烦恼。虽然这样的生活并不是我想要的，但是我相信现阶段努力不颓废，总会可以达到自己的目标。

# 回首

2020这一年已经彻底结束。回首瞭望，这一年什么都没有收获到，完全碌碌无为的一年，连跨年时都有点挫折。总的来说，2020是我彻底失败的一年;

# 展望

2021，不要再让你自己的生活一团糟了，不要再让别人为你担心了，不要一直一直停滞不前了。youself只要每次看到这段话，你就要反思自己，你的承诺兑现了吗，你有改变变优秀吗。现在此时此刻的你在敲这些字真的很悲惨，生活糟糠，工资低廉，无房无车，你真的落后别人很多很多了。2021年的开端，我在山顶度过的，看着这个城市的灯光辉煌，我的存在是那样微乎其微。我已经从22跨到了23，留给未来的机会又浪费了一年。 2021活好自己，改变自己，争取早日去上海，就俩字：挣钱

三年前决定你现在，你现在决定三年后，越留后面，压力越大;
